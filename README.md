# -# TECH ORDA Interactive Test
прпр
Интерактивный тест для студентов по теме Product and Project Management.

Проект представляет собой автономный HTML-файл с встроенным банком вопросов, вариантами ответов и логикой проверки. Тест можно использовать как учебный тренажёр или как демонстрационный тест внутри LMS-платформы.

## Основные возможности

- режим изучения с мгновенной обратной связью;
- режим тестирования со случайной выборкой вопросов;
- автоматический подсчёт правильных и неправильных ответов;
- отображение итогового результата в процентах;
- поиск по вопросам и вариантам ответов;
- фильтрация вопросов по статусу;
- адаптивный интерфейс для компьютера и мобильных устройств;
- работа без подключения к интернету;
- отсутствие внешних библиотек и серверной части.

## Назначение

Проект предназначен для организации самостоятельной подготовки студентов, промежуточного контроля знаний и размещения в LMS как интерактивного учебного ресурса.

## Технологии

- HTML
- CSS
- JavaScript

## Запуск

1. Скачайте файл проекта.
2. Откройте HTML-файл в браузере.
3. Выберите режим работы:
   - режим изучения;
   - режим теста.

## Использование в LMS

Файл можно загрузить в LMS как HTML-ресурс или встроить на страницу через iframe.

Пример:

```html
<iframe 
  src="Tech Orda тест.html" 
  width="100%" 
  height="900" 
  style="border: none;">
</iframe>
